# EMOF 3D Pipes Overlay

Map-anchored, perspective-projected 3D Pipes-style chart overlay for [Extensible Map Overlay Framework](https://mods.factorio.com/mod/extensible-map-overlay-framework).

## Usage

Enable **Extensible Map Overlay Framework** and this mod, open chart view, then toggle **3D Pipes** from **Chart Controls**.
